Neurochemical Mapping
9 Reward Pathway Predictions
Drop-Reset
Phasic dopamine craving–hit–baseline loop. No resolution phase. Structurally analogous to addiction cycling.
Continuous Ascent
Delayed gratification. Sustained anticipation with full resolution. Healthy goal-directed behavior pattern.
Intimate Narrative
Attachment and bonding circuitry. Oxytocin–dopamine overlap. Emotional salience over sonic climax.
Call-Response
Social reward circuitry. Partial reward sustaining anticipation. Interpersonal dialogue dopamine analog.
Burst / Compressed
Impulsivity analog. Immediate release, no anticipation phase. Distinct from cycling — no craving buildup precedes the hit.
Perpetual Groove
Motor-dopamine coupling. Basal ganglia rhythm entrainment. Synchrony reward. Lateral — not phasic.
Plateau / Sustained
Tonic dopamine. Flow-state analog. Sustained engagement without phasic cycling. Possibly meditative.
Gospel / Communal
Communal vocalization endorphins. Oxytocin via group synchrony. Emotional catharsis. Escalation without conventional resolution — transcendence.
Psychedelic / Improv
Novelty-seeking dopamine (reward of the unexpected). Serotonergic entrainment in intended chemical context. Architecture designed for altered neurochemical states — unique in dataset.

Clinical Parallel — Secondary Analysis
Country’s architectural migration from Type 3 to Type 1 occurred between 2008–2016. The opioid epidemic in rural white America accelerated between 2010–2017. Geographic, demographic, temporal, and architectural coincidence — the most clinically pointed secondary analysis the study can produce.

Type 9 — Architecture Predicts Substance
The Grateful Dead’s audience substance use was predominantly psychedelic rather than opioid or stimulant. If drop-reset cycling correlates with opioid/stimulant use and psychedelic architecture correlates with psychedelic use, the architecture-predicts-substance hypothesis is supported across two independent data points simultaneously. Type 9 makes that test possible. The Dead’s crowd was also a structural co-participant in the architecture — the only case in the dataset where the audience co-generates the form in real time.

Gospel — Secular Derivatives
Jackie WilsonEarliest bridge — gospel vocal architecture (melisma, escalation, breaking point) applied to secular R&B; influenced James Brown and Michael Jackson; makes downstream lineage traceable
Sam CookeGospel architecture translated intact into secular pop — You Send Me uses identical escalation mechanics to his Soul Stirrers recordings; proof the taxonomy is structural not genre-based
Otis ReddingTry a Little Tenderness: opens as Type 3, transforms into Type 8 within a single track — within-song architectural migration; proof of concept for the Spotify segment-level methodology
Al GreenReverse migration: secular Type 3/4 → ordained minister → pure Type 8; same voice traversing two architecture types across a documented biographical arc — rarest evidence in the dataset
Bill WithersNo gospel background — arrived at Type 8 architecture through emotional instinct alone. Lean on Me: verse as testimony, chorus as collective invitation. Lovely Day: longest sustained note in pop recording — the transcendence moment, frozen.

Contemporary Pop — Migration & Calibration
Taylor SwiftType 3 (2006–10) → Type 1 (2012–17) → Type 3 return (2020) — only artist of comparable scale to reverse a Type 1 migration voluntarily; Folklore / Evermore is the dataset’s most important contemporary data point
Katy PerryMaximally pure Type 1 calibration specimen (Roar, Firework, Teenage Dream) — empowerment lyrics, neutral drug content; ideal Cell B in the 2×2 lyrical design; the neuroimaging standard for uncontaminated drop-reset cycling
BeyoncéLemonade: Type 3 (Sandcastles) + Type 1 (Formation) + Type 4 (Freedom) + Type 2 (All Night) within one album; contemporary cross-taxonomy range approaching Beatles scale
Billie EilishType 1 subversion — builds anticipation, withholds the drop; operates in Type 7 with Type 1 surface features; bad guy tests what the dopaminergic loop does when the hit is deliberately withheld
AdeleContemporary Type 3 at stadium scale — demonstrates that patient earned-resolution architecture sustains mass commercial success in a Type 1 production environment
Ed SheeranShape of You: purest evidence that architecture is independent of sonic costume — drop-reset cycling in acoustic folk instrumentation
Cross-Taxonomy Case Study — The Beatles (Six of Nine Architecture Types)
Type 3 (Yesterday, Blackbird) · Type 2 (A Day in the Life — ascent sections) · Type 4 (I’ve Got a Feeling) · Type 5 (Helter Skelter) · Type 8 (Let It Be — gospel-adjacent) · Type 9 (Tomorrow Never Knows, I Am the Walrus, Revolution 9 — reclassified from Type 7 to Type 9: deliberate disorientation vs. sustained calm are architecturally distinct). Their structural evolution from 1962 to 1970 traces six of nine architecture types and makes them the most valuable cross-taxonomy validation case in the dataset.
